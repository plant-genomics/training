Bioinformatics tutorials

Priority topics currently under development are genome assembly, phylogenomics, variant calling, and RNA-seq.These will appear as items in the left hand panel.

Here are some links to resources that are particularly useful for getting started in plant bioinformatics:

  • R: Software carpentry tutorial: http://swcarpentry.github.io/r-novice-inflammation/
  • Git: Software carpentry tutorial: http://swcarpentry.github.io/git-novice/
  • Algorithms: Coursera:https://www.coursera.org/learn/dna-sequencing
  • Algorithms: Ben Langmead:https://www.youtube.com/user/BenLangmead
